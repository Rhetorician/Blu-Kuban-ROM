Anclaje a red de USB

El equipo puede compartir la conexión a Internet de su dispositivo si los ancla mediante un cable USB
  • El anclaje funciona con Windows 7, Windows Vista y Linux

  • No se puede conectar una tarjeta SD al equipo cuando está anclado

  • Para obtener más información, incluida información acerca de cómo utilizar el anclaje a red de USB con otros sistemas operativos, diríjase a http://www.android.com/tether#usb.

Modem Bluetooth

Puede conectar como modem su dispositivo Android a su equipo mediante una conexión Bluetooth, para compartir la conexión a Internet del dispositivo con su equipo
  • El modem Bluetooth funciona con cualquier equipo u otro dispositivo que pueda obtener conexión a Internet mediante Bluetooth

  • Para obtener más información, consulte http://www.android.com/tether#bluetooth
