Visualizzazione files

Mostra documenti

Premi e apri il file. E' possibile aprire files di diversi tipi solo se hai installato l'applicazione che permette di aprirli correttamente.

Mostra per modalità

Premi l'icona Griglia sulla barra degli strumenti (quella più a destra) per vedere i files o le cartelle nella modalità a Icone o Elenco.

Premi "Menu" -> "Impostazioni" -> "Mostra" e seleziona una delle opzioni tra Icone, Elenco e Dettagli.

Mostra per dimensione

Premi "Menu" -> "Impostazioni" -> "Mostra" -> "Dimensione" e seleziona una tra le opzioni Normale e Piccola.

Ordinamento files

Premi "Menu" -> "Impostazioni" -> "Mostra" -> "Ordinamento" e seleziona una tra le opzioni Nome, Tipo, Dimensione, Modificato e Casuale.

Apri come

E' possibile aprire un file come TESTO, IMMAGINE, etc premendo a lungo sul file e selezionando 'Apri come'. Se il file non corrisponde al tipo che hai selezionato allora si vedrà un contenuto inatteso del file.

Riproduzione file Video/Audio

Premi e apri il file. Al momento il tipi di formati supportati sono MP3, AMR, WAV, WMA, MP4, 3GP, WMV sia nella scheda SD che nella modalità remota. E' possibile provare diversi lettori multimediali.

Mostra immagini

Premi e apri l'immagine. Attualmente sono supportati i formati JPG, PNG, GIF, BMP sia nella scheda SD che nella modalità remota.

E' possibile impostare le immagini come sfondo o icone di contatto, vedere o impostare i dettagli di proprietà dell'immagine.

Mostra file nascosti

Per mostrare i files nascosti aggiungere un punto (.) davanti al file, premere "Menu" -> "Impostazioni" e poi togliere la spunta alla voce "Mostra file nascosti".
