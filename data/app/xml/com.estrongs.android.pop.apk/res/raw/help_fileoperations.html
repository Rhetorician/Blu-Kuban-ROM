File Operations

Select files

Multi-Select

Click the multiselect button(the second icon from the left to the right), then select several files. And the toolbar icons will change for operation.

Cut/Copy/Delete files

To operate these files, first select one or multiple files, then long press any file or folder(or press MENU/Operations), it will pop-up the operation dialogue, including the options of cut, copy, delete, Rename, and Properties, choose what you want. Or use the toolbar icon to do that.

For example:

  • Click MENU/Operations, pop up the operations dialogue.
  • Click cut, then go to the destination directory, click menu/Operations, select the paste option in the operations dialogue to put these files here.

To cancel the task, drag down the down arrow which is in the left top corner on the screen at the moment, you will see the task progress bar. Touch the progress bar, you can cancel this process.

To delete files or folders, long press some files or folder that you want to delete, pop-up the operation dialogue, click delete.

If clipboard is enabled in the settings, after you copied some files, they will be put on the clipboard, you can paste one or some of them at any folder, any time.

To clear clipboard, drag up or click the up arrow of screen in the middle of the bottom of the screen, long press what you want to clear up, choose the option of delete or delete all.

Clipboard toolbar

The toolbar of clipboard include four icons: Paste all, Paste, Minimize, Clear All. You can just quickly Operate by using the toolbar.

Send /Upload files

In Local Tab, long press a file. In the operations dialog, Click the send, choose one way such as Gmail, work mail, YouTube, messaging and Picasa(different options according to your file type and ROM version).

Edit text file

To new a text file, click MENU/New, input filename.txt in textbox , click ok.

To edit a text file, click and open file, input or modify the file content, click MENU/save.

Root permission operation

Root permission is an expermental feature in lab, it only fits for professional user with busybox installed. Normal user please don't use it, it might cause Force Close on some machine.

If you make sure you want to use it, click MENU/setting, select works as super user in Root Explorer section, follow the instructions in the pop-up box. Then enable Mount checkbox, now you can operate files in system directory. Disable mount checkbox, you will get the tips, "system directory mounted as read-only". From this time, you can only read system directory, cannot modify it.

Be careful! Any modification on system files might cause the unstablity of the system!

Set a folder as HOME

To set a folder as home directory, long press some folder you want set for HOME and choose Set As Home. It will take effect after you restart the File Explorer.
