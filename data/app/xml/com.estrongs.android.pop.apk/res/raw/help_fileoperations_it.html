Gestione files

Selezione files

Selezione multipla

Premi il tasto selezione multipla (la seconda icona da sinistra verso destra), poi seleziona alcuni files. In questo modo le icone della barra degli strumenti cambieranno per eseguire l'operazione.

Taglia/Copia/Elimina files

Per lavorare con i files prima seleziona uno o più files, poi premi a lungo su qualsiasi file o cartella (oppure premi "Menu" -> "Operazioni"). A questo punto verrà mostrata una nuova finestra con l'operazione da effettuare, incluse le opzioni Taglia, Copia, Elimina, Rinomina e Proprietà, in modo da selezionare quella che ti interessa. Altrimenti puoi sempre usare la barra degli strumenti per fare le stesse operazioni.

Per esempio:

  • Premendo "Menu" -> "Operazioni" viene mostrata la finestra con le operazioni disponibili.
  • Successivamente premi Taglia, poi vai nella cartella di destinazione, premi "Menu" -> "Operazioni", seleziona l'opzione Incolla nella finestra delle operazioni per inserire i diversi files.

Per annullare l'attività trascina giù la barra delle notifiche (una freccetta rivolta verso il basso mostra il progresso dell'attività) e poi tocca la barra per annullare l'operazione in corso.

Per eliminare i files o le cartelle, premi a lungo su qualsiasi file o cartella che vuoi eliminare. A questo punto verrà mostrata una finestra con l'operazione da effettuare e poi premi Elimina.

Se gli appunti sono stati abilitati nelle impostazioni, dopo che si saranno copiati alcuni files questi saranno inseriti negli appunti e pronti per essere copiati su qualsiasi cartella in ogni momento.

Per pulire gli appunti, trascina in su o premi la freccia rivolta verso l'altro a metà schermo. Poi premi a lungo sugli oggetti disponibili e scegli l'opzione Elimina o Elimina Tutto.

Barra strumenti appunti

La barra strumenti degli appunti include quattro icone: Incolla Tutto, Incolla, Nascondi Appunti, Elimina Tutto. E' possibile operare in maniera molto veloce usando questa barra degli strumenti.

Invia/Carica files

Nella scheda Locale premi a lungo su un file. Nella finestra operazioni premi il tasto 'Invia', scegli il metodo di invio come Gmail, Posta, YouTube, Messaggi e Picasa (sono previsti diversi metodi a seconda del tipo file e della versione della propria ROM).

Modifica file di testo

Per creare un nuovo file di testo premi "Menu" -> "Nuovo" -> "Files", inserisci 'nome_file.txt' nella casella di testo e poi premi OK.

Per modificare un file di testo premi a lungo sul file, poi 'Apri come', seleziona Testo come tipo file ed infine inserisci o modifica il contenuto del file. Per salvarlo premi "Menu" -> "Salva".

Utilizzo permessi di 'root'

I permessi di root è una funzione sperimentale ed è riservata solo agli utenti avanzati che hanno installato busybox. Gli utenti normali non hanno bisogno di questa funzione visto che potrebbe causare un BLOCCO TOTALE del sistema.

Se sei sicuro di volerla usare allora premi "Menu" -> "Impostazioni" -> "Opzioni di root" -> "Sfoglia come root" seguendo le indicazioni della finestra. Poi abilita la voce "Monta File System" in modo da operare sui files delle cartelle di sistema. Se si disabilita l'opzione di mount verrà mostrato il suggerimento 'Directory di sistema montata come sola lettura' e da questo momento in poi potrai soltanto leggere le cartelle di sistema e non modificarle.

Fai attenzione! Qualsiasi modifica sui file di sistema potrebbe causare installabilità al sistema!

Imposta cartella come PREDEFINITA

Per impostare una cartella come predefinita premi a lungo sulla cartella di interesse e poi scegli 'Imposta come predefinita'. La modifica sarà attiva solo dopo aver riavviato ES Gestore File.
