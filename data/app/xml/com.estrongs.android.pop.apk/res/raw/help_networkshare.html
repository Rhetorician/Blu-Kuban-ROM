Visit Network Files

The feature of Remote in ES File Explorer can manage your files in LAN/FTP/Bluetooth. You can look through, transfer and search files in one and more PCs.

New a Server

  1. Click Menu/New/Sacn, in Remote
  2. Or click Search on toolbar to scan
  3. Or input IP of your PC, for e.g. 192.168.1.100
  4. Input username/password, u/p is what you use to logon on the PC.
  5. If new a FTP server, sometime you need change the charset encoding and active/passive mode to match your ftp server.
  6. Click OK.

Click Server to browse its files. Long press the server icon and here you can delete, rename, new sever,view server properties.

File Operations/Search/View

Operate files between your phones and your computer, such as copy, cut, delete, and rename files

The process is similar as what you've done on SDcard, refer to File Operations

Set a Remote Folder as Server

To set a remote folder as server, long press the folder and choose the option of Add to Server List
