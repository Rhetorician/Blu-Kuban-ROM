Introduzione

ES Gestore File è un completo gestore di file locali/LAN/FTP/Bluetooth, sicurezza, applicazioni, processi e segnalibri.

Alcune delle funzioni includono:

  • Operazioni con file locali, LAN, FTP e Bluetooth
  • Riproduzione file audio e video in locale o da remoto
  • Chiusura processi per liberare memoria
  • Widget auto chiusura processi
  • Protezione applicazioni
  • Ritrovamento telefono perduto
  • Salvataggio SMS e contatti
  • Modifica file di testo/ini
  • Visualizzazione o meno dei file nascosti
  • Analisi spazio utilizzato nella scheda SD
  • Visualizzazione info stato batteria
  • Operazioni multiple con files/applicazioni e un potente gestore appunti
  • Compressione/scompattamento file zip
  • Visualizzazione file o altri file aperti con programmi di terze parti
  • Modifica vista e ordinamento
  • Ricerca file su scheda SD, condivisioni e Bluetooth
  • Invio file ad altre persone
  • Salvataggio applicazioni, (dis)installazione e creazione collegamenti
  • Selezione immagini come sfondo o icona contatti
  • Selezione cartella locale come principale
  • Selezione cartella remota come server
  • Accesso a file/cartelle come utente root
