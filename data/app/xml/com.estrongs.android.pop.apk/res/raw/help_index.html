Introduction

ES File Explorer is a featured local/LAN/FTP/Bluetooth file & security & application & task & bookmark manager.

Some of the features include:

  • Local, LAN, FTP and Bluetooth file operations
  • Play audio or video files locally or remotely
  • Kill process to free memory
  • Auto-kill Widget
  • Protect application
  • Find your lost phone
  • Backup SMS and contacts
  • Edit text/ini file
  • Show/hide hidden files
  • Analysis use of space on SD card
  • Show battery life information
  • Multi-operate on files/apps and powerful clipboard
  • zip/unzip the zip files
  • View files or open other files with the third party programs
  • Change Views and Sort
  • Search files on sd card, shares and Bluetooth
  • Send files to others
  • App backup & (un)install & make shortcut
  • Set pictures as wallpaper or contact icon
  • Set folder as home in local
  • Set folder as server in Remote
  • Root explorer
