Condivisioni di rete

La funzione remota in ES Gestore File permette di gestire i files tramite LAN/FTP/Bluetooth. E' possibile vedere, trasferire e cercare i files tra uno o più computer.

Nuovo Server

  1. Seleziona "Menu" -> "Nuovo" -> "Scansione" nella modalità remota
  2. Altrimenti seleziona "Cerca" nella barra degli strumenti per fare la scansione
  3. Oppure inserisci l'indirizzo IP del computer, ad esempio 192.168.1.100
  4. Inserisci nome utente/password (i dati necessari per connettersi al computer).
  5. Se si crea un nuovo server FTP a volte è necessario modificare il tipo di codifica ed impostare la modalità attiva/passiva in modo che entrambe corrispondano.
  6. Ora seleziona OK.

Seleziona il server per sfogliare il suo contenuto. Premi a lungo l'icona del server per eliminare, rinominare, creare un nuovo server o vedere le sue proprietà.

Gestione files/Cerca/Vista

Permette di lavorare con i files presenti nei telefoni e nei computer con azioni come Taglia, Copia, Elimina e Rinomina.

Il metodo è simile a quello utilizzato con i dati presenti nella scheda SD, vedi Gestione files.

Imposta cartella remota come server

Per impostare una cartella remota come server si deve premere a lungo la cartella e poi scegliere l'opzione 'Aggiungi alla lista dei server'.
