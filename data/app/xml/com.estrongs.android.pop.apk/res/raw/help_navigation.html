Navigation

You can find the following components in the main window:

Tab

Local: Show files in your phone.

LAN: Show files in your LAN Share.

FTP: Show files in FTP Server.

Bluetooth: Show files in Bluetooth devices.

Address bar

Below top Tab, shows current address (such as /sdcard/movie).

There's a button on the leftmost of address bar when tabs are hidden.

Click it to switch between the tabs.

There's a button on the rightmost of address bar.

Click it to switch open or close the toolbar.

And click ADDRESS BAR to show history path.

Toolbar

- Press the icons on the toolbar:

Home(house): Switch Home and Root directory

Multi(plus): Multi-Select mode

Search(glasses): Search files

Up(folder): Up a level, double-click exit

help(Question mark): Icon meaning, try slide:)

Layout(grid): Change view among Icon,List,Detail

- Long-press the toolbar icons:

Home: Show ES Bookmark

Multi: Select All or Select None

Search: Sort by

Up: Exit

App: Choose among managers

Layout: Use large/small icons in the list
